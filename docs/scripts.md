# Auto-Hub Scripts
Version: v0.1 (Manual)

## Overview

This document contains message templates for manual AutoHub outreach.

Goal:
- contact sellers
- contact buyers
- connect with garages
- handle common questions

Principles:
- keep messages short
- sound like a real person
- do not oversell
- do not use "startup" language
- focus on usefulness

---

## Seller

### First message

```
Hey, I’m putting together a small list of good cars under $15k in Toronto and trying to filter out scams and bad listings.
Your car looks solid - can I include it?
```

### If seller agrees

```
Nice, thanks.

I also sometimes help buyers connect with a mechanic before purchase so deals go smoother.
If I bring someone serious, would that be okay?
```

### If seller asks what this is

```
Nothing complicated — I’m just testing a manual process right now.
I’m collecting decent listings and helping buyers avoid bad deals.
```

### If seller asks for website
```
No full site yet — I’m testing it manually first to make sure it actually helps people.
```

### If seller asks if it costs money

```
No, not at this stage.
Right now I’m just testing the process.
```

### If seller is unsure or suspicious

```
No worries, just thought your listing looked solid.
If you change your mind later, feel free to reach out.
```

### Follow-up (if no reply)

```
Hey, just checking once more — are you okay with me including your listing if I find a serious buyer?
```

### Exit

```
All good, thanks anyway.
```

## Buyer

### First message

```
Hey, I’m putting together a clean list of cars under $15k in Toronto and filtering out obvious scams and weak listings.
If you’re looking, I can send you a few decent options.
```

### If buyer is interested

```
Sure — what are you looking for?
Type of car and budget range?
```

### When sending options

```
I picked a few solid options for you.
These look cleaner than average, but I’ll flag any risks I notice.
```

### If buyer asks how it works

```
I filter listings, send better options, and if needed help arrange a mechanic check before purchase.
```

### If buyer asks if it’s free

```
For now, yes.
I’m still testing it manually.
```

### If buyer asks about trust

```
I don’t guarantee anything — I just filter out obvious risks.
For anything serious, inspection is always the next step.
```

### Offer inspection

```
If you like one of these, I can help you get it checked before buying.
```

### Follow-up

```
Hey, just checking — still looking or already found something?
```

### Exit

```
No problem.
If you start looking again, feel free to message me.
```

## Garage

### First message

```
Hey, I’m working with people buying used cars and helping them check vehicles before purchase.
Do you guys do pre-purchase inspections?
```

### If garage says yes

```
Nice.
I’m putting together a small list of shops I can send buyers to.
If I send someone over, are you okay with that?
```

### If garage asks what this is

```
I’m testing a simple process where I help buyers find better listings and connect them with inspection shops.
```

### If garage asks for website

```
No full platform yet — I’m testing it manually first.
```

### If garage asks about payment

```
Not at this stage.
I’m just connecting buyers with good shops.
```

### If garage is interested

```
Perfect.
What types of cars do you usually work on?
And do you offer same-day inspections?
```

### Follow-up

```
Hey, just checking — are you open to inspections if I send someone?
```

### Exit

```
No problem, thanks for the reply.
```

## Common Questions

### Why are you doing this?

```
Because the used car market is messy, and it’s hard to find decent listings without wasting time or risking a bad deal.
```

### Are you a dealer?

```
No, I’m not a dealer.
I just filter listings and help people avoid weak or risky deals.
```

### Can you guarantee the car?

```
No.
I can reduce obvious risk, but inspection is always needed before buying.
```

### How do you make money?

```
Right now I’m just testing the process.
Focus is on whether it actually helps people.
```
